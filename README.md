# 🍕 Pizza Sales Analysis | MySQL + Power BI

## 📌 Overview

This project analyzes pizza sales data using **MySQL** and **Power BI** to uncover business insights. The dataset was imported into MySQL, where SQL queries were used to answer business questions related to sales, revenue, customer ordering patterns, and product performance. The results were then visualized in an interactive Power BI dashboard and summarized in a presentation.

---

## 📂 Dataset

The dataset contains information about pizza orders, including:

- Order details
- Pizza name
- Category
- Size
- Quantity
- Price
- Order date and time

**Dataset Highlights**
- ~21,000 orders
- ~48,000 pizzas sold
- Multiple pizza categories and sizes

---

## 🛠️ Tools Used

- **MySQL** – Data storage and SQL analysis
- **Power BI** – Interactive dashboard creation
- **Microsoft PowerPoint** – Project presentation

---

## 🚀 Project Workflow

1. Imported the dataset into MySQL.
2. Created database tables.
3. Wrote SQL queries to answer business questions.
4. Analyzed sales, revenue, and customer ordering trends.
5. Connected MySQL with Power BI.
6. Built an interactive dashboard.
7. Summarized findings in a PowerPoint presentation.

---

## 📊 Dashboard

The Power BI dashboard includes:

- Total Revenue
- Total Orders
- Total Quantity Sold
- Average Order Value
- Revenue by Category
- Quantity by Pizza Size
- Top Selling Pizzas
- Monthly Revenue Trend
- Daily Revenue Trend
